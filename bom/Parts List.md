# Parts List (Japan Version)

The following parts were used to assemble the modified UMI Hand at Nakamura Laboratory.

## Linear Rail

### MGN9C Linear Guide (MGN9C1R150Z0HM, MISUMI)

https://jp.misumi-ec.com/vona2/detail/221005481777/
- ブロック種類：標準タイプ
- レール種類：標準
- ブロック数：1
- レール寸法：9
- 組み立て寸法：10
- レール長さ：150

## Spring

### Compression Spring (UR10-50)

https://jp.misumi-ec.com/vona2/detail/110302604820/
- 自由長：50
- 外径：10
- 線形：0.8

## Notes

The primary difference from the official UMI Hand BOM is the linear rail system.

To accommodate components that are readily available in Japan, the following CAD files were modified:

- gripper-bottom_plate
- gripper-grip

All other components follow the original UMI Hand design.
