# 3D Printing Settings

The following settings were used to print the modified UMI Hand components at Nakamura Laboratory.

## Printer

- Printer: Bambu Lab P2S
- Nozzle: 0.4 mm
- Material: PLA

## Recommended Settings

| Parameter | Value |
|------------|------------|
| Layer Height | 0.20 mm |
| Initial Layer Height | 0.20 mm |
| Wall Loops | 3 |
| Top Shell Layers | 6 |
| Bottom Shell Layers | 6 |
| Infill Density | 20% |
| Infill Pattern | Gyroid |
| Supports | Enabled where necessary |
| Build Plate Adhesion | Default |
| Nozzle Temperature | 200 |
| Bed Temperature | 60 |

## Notes

- STL files provided in this repository were successfully printed using a Bambu Lab P2S with a 0.4 mm nozzle.
- PLA was sufficient for assembly and functional testing.
- Total printing time for all modified parts is less than 12 hours on the P2S.

## Material Validation

At the time of writing, all printed components were manufactured using PLA and successfully assembled into a functional UMI Hand.

TPU materials have not yet been tested. Users who intend to print any components in TPU should perform additional validation regarding dimensional accuracy, assembly compatibility, and mechanical performance.
